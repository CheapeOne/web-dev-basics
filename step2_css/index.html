CSS: Adding style

Here's a styled up html page. You style html using CSS, adding colors, margins, animations. The CSS in this case is right in the "style" tag of this html. It's nested below the "head" tag, which normally holds any metadata about the page. It's placed here so that the browser reads it first before trying to display any of the content within the "body" tag.

(check the title of this page on the tab in your browser, it's the same as the title tag in the head!)
